Tutorial Quick Start

Server Langkah 1: Unduh dan Instal Cloudflared

Unduh versi Cloudflared yang sesuai dengan sistem operasi Anda dari halaman release.

Ikuti petunjuk instalasi yang muncul di layar untuk menyelesaikan pemasangan.

Server Langkah 2: Unduh LoginMyID

Unduh aplikasi LoginMyID dan ekstrak di komputer. Jalankan run.bat, kemudian perhatikan domain sementara yang keluar dari cloudflared untuk digunakan sebagai domain koneksi dari client.

Client Langkah 1: Unduh Nekoray

Unduh aplikasi Nekoray di client yang akan koneksi ke server dan ekstrak di komputer.

Client Langkah 2: Hubungkan ke Server

Masukkan Link XRay Baru di aplikasi Nekoray. Kemudian edit koneksinya, terutama pada bagian domain diubah sesuai dengan domain yang keluar di cloudflared server.

Client Langkah 3: Mulai Mengakses

Setelah perangkat terhubung, Anda dapat mulai mengaksesnya dari mana saja melalui aplikasi LoginMyID.
